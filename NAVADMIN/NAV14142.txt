NARR/REF A IS MICROSOFT PRODUCT LIFECYCLE ANNOUNCEMENT FOR END OF LIFE 
SUPPORT FOR WINDOWS SERVER 2003.

RMKS/1.  THIS NAVADMIN PROVIDES GUIDANCE IN REGARDS TO THE END OF SERVICE 
SUPPORT FOR MICROSOFT WINDOWS SERVER 2003 PRODUCTS.  PER REFERENCE (A), 
EFFECTIVE 14 JULY 2015, MICROSOFT WILL NO LONGER PROVIDE VENDOR SUPPORT 
(AUTOMATIC FIXES, UPDATES, SECURITY PATCHES OR ONLINE TECHNICAL ASSISTANCE) 
FOR:
    A.  WINDOWS SERVER 2003, DATACENTER EDITION (32-BIT X86)
    B.  WINDOWS SERVER 2003, DATACENTER EDITION FOR ITANIUM-BASED SYSTEMS
    C.  WINDOWS SERVER 2003, DATACENTER X64 EDITION
    D.  WINDOWS SERVER 2003, ENTERPRISE EDITION (32-BIT X86)
    E.  WINDOWS SERVER 2003, ENTERPRISE EDITION FOR ITANIUM-BASED SYSTEMS
    F.  WINDOWS SERVER 2003, ENTERPRISE X64 EDITION
    G.  WINDOWS SERVER 2003, STANDARD EDITION (32-BIT X86)
    H.  WINDOWS SERVER 2003, STANDARD X64 EDITION
    I.  WINDOWS SERVER 2003, WEB EDITION

2.  PROGRAM MANAGERS IN COORDINATION WITH THEIR ECHELON II COMMAND 
INFORMATION OFFICER (CIO), COMPTROLLER AND RESOURCE SPONSOR SHALL MIGRATE ALL 
NAVY INFORMATION TECHNOLOGY (IT) SYSTEMS OR NETWORK INFRASTRUCTURE ASHORE AND 
TACTICAL (AFLOAT, ALOFT) OPERATING ANY OF THE WINDOWS SERVER 2003 
ENVIRONMENTS LISTED IN PARAGRAPH ONE, TO A WINDOWS SERVER 2008 OR GREATER 
ENVIRONMENT NOT LATER THAN
14 JULY 2015.  IT SYSTEMS INCLUDE PROGRAMS AND NON-PROGRAMS OF RECORD, 
EXCEPTED LEGACY NETWORKS, NEXT GENERATION ENTERPRISE NETWORK, OUTSIDE THE 
CONTINENTAL UNITED STATES NAVY ENTERPRISE NETWORK, RESEARCH DEVELOPMENT TEST 
AND EVALUATION, OFFICE OF NAVAL INTELLIGENCE, U.S. NAVY BUREAU OF MEDICINE 
AND SURGERY, TRAINING NETWORK, AND NAVAL EDUCATION TRAINING COMMAND NETWORKS 
USING WINDOWS SERVER 2003 OPERATING SYSTEMS.

3.  PROGRAM MANAGERS MUST SUBMIT A WAIVER REQUEST VIA THEIR ECHELON II CIO TO 
DEPUTY DEPARTMENT OF THE NAVY CHIEF INFORMATION OFFICER - NAVY (DDCIO(N)) NOT 
LATER THAN 24 JULY 2014 IF THEY ANTICIPATE NOT MIGRATING TO WINDOWS SERVER 
2008 OR GREATER BEFORE 14 JULY 2015.
WAIVER DETAILS WILL BE PROVIDED VIA DDCIO(N) UNDER SEPARATE COVER.
PROGRAM MANAGERS ARE REMINDED NOT TO NEGOTIATE EXTENDED SUPPORT CONTRACTS 
WITH VENDORS FOR WINDOWS SERVER 2003 SUPPORT, ANY EXTENDED LIFE SUPPORT 
REQUIREMENTS WILL BE PROMULGATED BY DDCIO(N) AND NEGOTIATED BY THE 
APPROPRIATE NAVY PROGRAM EXECUTIVE OFFICE.

4.  POINT OF CONTACT IS MICHAEL CRICCHIO, OPNAV N2N6BC2, AT
(571) 256-8510 OR MICHAEL.CRICCHIO1(AT)NAVY.MIL.
