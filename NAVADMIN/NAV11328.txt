RMKS/1.  THIS NAVADMIN ANNOUNCES AN INTERIM APPLICATION PROCESS FOR THE 
ENLISTED SUPERVISOR RETENTION PAY (ESRP) PROGRAM.

2.  ESRP-ELIGIBLE SAILORS AND COMMAND CAREER COUNSELORS SHOULD REVIEW REFS A 
AND B, WHICH PROVIDE FURTHER PROGRAM DETAILS.  REFS ARE POSTED ON THE NUCLEAR 
ENLISTED COMMUNITY MANAGER WEBSITE AT:
WWW.PUBLIC.NAVY.MIL/BUPERS-NPC/ENLISTED/COMMUNITY/NUCLEAR/.

3.  THE OFFICER PERSONNEL INFORMATION SYSTEM (OPINS) FORCE MANAGEMENT SYSTEM 
(FORMAN) IS BEING REPROGRAMMED TO SUPPORT ESRP ZONE RESTRUCTURE.
CHANGES TO OPINS FORMAN WILL NOT BE COMPLETE WHEN REFS A AND B BECOME 
EFFECTIVE 01JAN12.  BEGINNING 01JAN12, PARA 4 BELOW REPLACES PARA 10 OF REF B.  
ORIGINATOR WILL CANCEL THIS MESSAGE UPON COMPLETION OF OPINS PROGRAMMING.

4.  APPLICATION PROCESS:
    A.  SUBMIT ESRP REENLISTMENT REQUESTS 35-120 DAYS IN ADVANCE OF THE 
MEMBER'S DESIRED REENLISTMENT DATE TO ENSURE TIMELY PROCESSING AND APPROVAL.  
THE ESRP PROGRAM MANAGER WILL REVIEW AND APPROVE EXCEPTIONS TO THIS 
REQUIREMENT ON AN INDIVIDUAL BASIS.
    B.  ESRP REQUESTS MUST BE SENT VIA E-MAIL TO BOTH 
"NXAG_N133D3(AT)NAVY.MIL" AND "NXAG_N133D2(AT)NAVY.MIL".  INCLUDE IN THE E-
MAIL REQUEST A SCANNED COPY ENCL 1 OF REF A AND THE UNIT IDENTIFICATION CODE 
(UIC) OF THE REQUESTING COMMAND.  IF OPERATIONAL COMMITMENTS PRECLUDE 
SUBMITTING ESRP APPLICATIONS VIA EMAIL, SUBMIT THE REQUEST VIA MESSAGE TO THE 
COMMAND'S IMMEDIATE SUPERIOR IN CHARGE (ISIC).  THE ISIC WILL THEN SUBMIT 
REQUEST VIA E-MAIL.
    C.  THE ESRP PROGRAM MANAGER WILL ACKNOWLEDGE RECEIPT OF THE ESRP REQUEST 
VIA E-MAIL.
    D.  UPON PROCESSING, NAVY PERSONNEL COMMAND PERS 811 WILL SEND A MESSAGE 
TO THE REQUESTING UIC WITH THE STATUS OF THE REQUEST.

5.  POINT OF CONTACT:
    - NUCLEAR ENLISTED PROGRAM MANAGER, N133D COMM:
(703)695-1276/3301/4449/DSN 225 OR E-MAIL AT NXAG_N133D3(AT)NAVY.MIL, 
NXAG_N133D2(AT)NAVY.MIL.
    - COMMANDS AND UNITS UNABLE TO ACCESS THE LISTED WEBSITE MAY CONTACT POC'S 
TO OBTAIN REF A.

6.  RELEASED BY VICE ADMIRAL S. R. VAN BUSKIRK, N1.//
